선물함 상태 설명 도움말

문 상태 : 선물함 문의 열림 또는 닫힘 상태를 표시
상품여부 : 선물함 안에 상품이 있는지 표시
보관상품 종류 : 해당 선물함 번호에 실제 보관하려는 상품 설정 가능
상품보관 수동설정 : 상품의 보관 유무를 수동으로 설정 가능
비밀번호 변경 : 사용자가 비밀번호를 잊어버리거나 문이 닫혔을 경우 해당 선물함 오픈 비밀번호 초기화
